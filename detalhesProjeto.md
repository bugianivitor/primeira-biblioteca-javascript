# Página de artigos Alura

## Problemas

### Problema 1 - Palavras repetidas

Quando se escreve artigos, algumas palavras se repetem muitas vezes. O programa precisa identificar as palavras que se repetem.

Isso facilitaria revisar os artigos escritos.

## Anotações pessoais

*Biblioteca:* pacotes de códigos reutilizáveis que resolvem um problema ou necessidade. Hoje a biblioteca é uma facilitadora e tem bibliteca para fazer:
* Validação
* Autenticação
* Testes
* Conexão com Banco de Dados
* Conexão com diversos outros serviços
* Animações no front end
* Conversão de arquivos

Podemos reaproveitar o que outras pessoas já desenvolveram e testaram.
Além das bibliotecas nativas, temos bibliotecas desenvolvidas por empresas ou comunidades.
Qualquer pessoa pode criar e publicar a biblioteca desde que os requisitos sejam atendidos.
